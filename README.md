# ⚽ Bet Bot v2

Минималистичный Telegram-бот для прогнозов на матчи.

## Что делает

- 📥 Принимает прогнозы вручную через `/add` (любой формат, главное чтобы было время `HH-MM` или `HH:MM`)
- 🤖 Автоматически читает прогнозы из заданного Discord-канала
- 🔔 Шлёт уведомление за **30 минут** и за **5 минут** до начала матча
- 🗑 Автоматически удаляет матч из списка через **5 минут после его старта**

---

## Переменные окружения

### Обязательные

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен TG-бота от @BotFather |

### Опциональные (для авто-парсинга Discord)

| Переменная | Описание |
|---|---|
| `DISCORD_TOKEN` | Токен Discord-бота из Developer Portal |
| `DISCORD_CHANNEL_ID` | ID канала с прогнозами (ПКМ на канал → Copy Channel ID) |
| `DISCORD_TARGET_TG_CHAT_ID` | Куда отправлять прогнозы из Discord (твой TG chat_id или ID группы) |

### Дополнительные

| Переменная | По умолчанию | Описание |
|---|---|---|
| `DEFAULT_TZ_OFFSET` | `3` | UTC+N для новых пользователей |
| `DATA_DIR` | `./data` | Папка для хранения JSON |

## Деплой на Railway

1. Залить код в GitHub
2. Railway → New Project → Deploy from GitHub
3. Добавить переменные окружения (хотя бы `BOT_TOKEN`)
4. Если нужен Discord — настроить три переменные про Discord

---

## Создание Discord-бота

1. https://discord.com/developers/applications → New Application
2. Bot → **включи "Message Content Intent"** (это критично — без него бот не видит текст сообщений)
3. Reset Token → скопируй
4. OAuth2 → URL Generator → выбери `bot` + права `View Channel`, `Read Message History`
5. Перейди по ссылке → добавь бота на сервер друга

## Команды бота

| Команда | Действие |
|---|---|
| `/start` | Приветствие и список команд |
| `/add` | Добавить прогнозы вручную |
| `/list` | Показать активные прогнозы |
| `/delete <id>` | Удалить один прогноз |
| `/clear` | Очистить все |
| `/settings` | Настройки (часовой пояс) |

---

## Формат прогнозов

Главное — чтобы в тексте было время в формате `HH-MM` или `HH:MM`. Всё остальное — любой текст.

**Несколько прогнозов отделяй пустой строкой:**

```
Soccer. Brazil. Acreano U20. 2-00
Santa Cruz Acre U20 — Independencia FC U20
ф1-4,5

Soccer. Australia. League 1 U20. 11-00
Hurstville U20 — Central Coast Mariners U20 п2 4+
```

Бот склеит каждый блок в одну строку и пришлёт уведомление в исходном виде.
